Calibre Readings Lists

Website and mobile app to manage reading lists using books from calibre e-book manager

How it will work?

  • A backend service will get data from Calibre Web RSS feed such as http://ebooks.tollkuci.com:8080/opds/navcatalog/4f6e6577657374. The data will be cached in some intermediate layer.
  • Web (responsive) UI will allow the user to create account/sign in, connect with the calibre UI and then create reading lists from their books. Also they will be able to manage their progress.
  • Mobile app will also be developed that will use REST services in server and provide similary features as the website.

Technical details

  • Website will be developed using ASP.NET MVC and pure.css frameworks.
  • Mobile apps will be developed using Ionic Framework
  • REST services to be decided
  • Database to be decided
  • To work on the project you will need Visual Studio 2015 and Ionic 2 apps for Visual Studio (check out https://taco.visualstudio.com/en-us/docs/tutorial-ionic2/ )
